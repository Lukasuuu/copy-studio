## 🎯 O que é?

**Copy Studio** é uma ferramenta profissional para desenvolvedores, copywriters e criadores de conteúdo que precisam transformar textos robóticos ou gerados por IA em copy **natural, persuasiva e com personalidade**.

Ideal para:
- 📄 **Landing Pages** — headlines, CTAs, bullets, hero sections
- 📱 **App Store / Play Store** — descrições, screenshots text, release notes
- 💬 **Onboarding de Apps** — microcopy, empty states, notificações push
- 📧 **E-mail Marketing** — campanhas de retenção, recuperação de carrinho
- 🎮 **Storytelling Interativo** — diálogos de jogos, narrativas gerativas

## ✨ Funcionalidades

### 🎨 13 Tons de Voz
Escolha o tom perfeito para o seu público:

| Tom | Perfil |
|-----|--------|
| 💬 **Conversational** | Como uma conversa de café |
| 💼 **Professional** | Corporativo, mas acessível |
| ⚡ **Persuasive** | Copy que vende |
| 😂 **Humorous** | Leve e divertida |
| ❤️ **Emotional** | Profunda, conecta no coração |
| 🎓 **Academic** | Formal, vocabulário elevado |
| 📰 **Journalistic** | Clara, objetiva, factual |
| 🎨 **Creative** | Poética, metafórica |
| 💻 **Technical** | Precisa, terminológica |
| 📈 **Marketing** | Hype, benefícios, CTA |
| ☕ **Casual** | Descontraída, gírias leves |
| 📖 **Storytelling** | Narrativa, gancho, arco |
| 📊 **Analytical** | Baseada em dados, lógica |

### 🖊️ 5 Estilos de Escrita
- **Formal** — sem contrações, linguagem completa
- **Casual** — com contrações, linguagem natural
- **Creative** — variações livres e expressivas
- **Technical** — precisa e terminológica
- **Marketing** — focada em conversão

### ⚡ 4 Níveis de Intensidade
| Nível | Efeito | Cor |
|-------|--------|-----|
| 🟢 **Light** | Toques sutis | Verde |
| 🔵 **Medium** | Equilíbrio natural | Azul |
| 🟠 **Aggressive** | Mudanças significativas | Laranja |
| 🔴 **Ninja** | Loop de 5 passes, máxima transformação | Vermelho |

### 📊 Human Score
Algoritmo interno que estima o quão "humano" o texto resultante está (0-100), baseado em:
- Uso de contrações (don't, I'm, etc.)
- Comprimento médio das frases
- Variedade lexical
- Presença de linguagem formal excessiva
- Elementos informais e expressivos

```
│   │   └── Header.tsx            ← Cabeçalho com logo neon

│   ├── main.tsx            ← Entry point
│   └── index.css           ← Estilos globais (glass, neon, gradients)
├── vercel.json             ← Config Vercel (deploy automático)
├── tailwind.config.js      ← Tema custom (cores neon)
└── package.json
```

### 🧠 Como funciona a Engine (`humanizer.ts`)

1. **Tone Replacement** — substitui palavras formais por alternativas naturais conforme o tom escolhido
2. **Style Transformation** — expande ou contrai contrações (formal vs. casual)
3. **Intensity Rules** — aplica regras de reescrita progressivas:
   - *Light*: remove advérbios intensificadores
   - *Medium*: substitui jargon corporativo
   - *Aggressive*: simplifica expressões formais longas
   - *Ninja*: quebra frases longas, varia ritmo, contrai tudo
4. **Human Score Estimation** — analisa métricas do texto resultante

## 🔄 CI/CD + Deploy Automático

### GitHub Actions
A cada `push` para `main` ou `develop`, e a cada **Pull Request**:
1. ✅ Checkout do código
2. ✅ Instalação de dependências com cache
3. ✅ Type check (`tsc --noEmit`)
4. ✅ Build de produção (`vite build`)
5. ✅ Verificação se `dist/` foi gerado

### Vercel
- **Deploy Preview** em cada push para branches
- **Deploy Production** automático ao mergear para `main`
- Framework preset: **Vite**
- Output directory: `dist`
